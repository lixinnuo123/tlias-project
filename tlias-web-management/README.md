# Tlias 教学管理系统

## 📌 项目介绍
本项目为黑马程序员Tlias教学管理系统实战项目，采用主流Java后端技术栈实现，涵盖部门管理、员工管理等核心功能模块。目前正在持续开发中。

## 🛠️ 技术栈
- **Java 8** ｜ **Spring Boot** ｜ **MyBatis**
- **MySQL** ｜ **PageHelper**（分页插件）
- **Lombok** ｜ **Maven** ｜ **Git**

## 🚀 如何运行
1. 执行项目根目录下的 `tlias.sql` 创建数据库表
2. 修改 `application.yml` 中的数据库用户名和密码
3. 启动类：`TliasApplication.java`
4. 访问接口：`http://localhost:8080`
